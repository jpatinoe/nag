# Nag 

A personal AI agent that remembers what you need to do and actually reminds you.

## The problem
Most task apps require you to open them, maintain them, and remember to check them. Nag works the other way: you tell it things casually, and it reaches out to you at the right time.

## How it works
1. Send Nag a message — as messy as you want
2. It figures out what needs a date, what's urgent, and what it needs to ask you
3. It reminds you via SMS or WhatsApp when the time is right

## Status
Work in progress. Currently building the task parsing layer.

## Tech stack
- Python
- Anthropic Claude API
- SQLite
- Twilio (SMS/WhatsApp)
